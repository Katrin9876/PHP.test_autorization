Для теста данного проекта необходимо в файле connect.php вместо текущих значений(приведены ниже)
указать данные своего сервера и базы данных, где будут храниться значения.
    $HOSTNAME='localhost';
    $USERNAME='root';
    $PASSWORD='******';
    $DATABASE='usersdb';
В данном проекте используется база данных 'userdb', в которой создана таблица 'registration'. 
Создать данную таблицу можно, запустив файл 'dbcreate.php' на своем сервере.
